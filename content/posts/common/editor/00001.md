## Editor設定
Atom -> Preference -> Editor
  - Font Family : Ricty Diminished  
  - Show invisibles : chek on  
  - Soft Wrap : chek on  

### テーマ設定
  - UI Theme : Atom Light
  - Syntax Theme : Atom Light

### Vim
  - vim-mode-plu
  - ex-mode

### ショートカットキー
Atomメニュー -> keymap

```
'atom-text-editor':
  'alt-shift-i': 'editor:auto-indent'
```

- 「key binding resolver」
  入力したショートカットキーがことで使われているかフッターに表示してくれる。
