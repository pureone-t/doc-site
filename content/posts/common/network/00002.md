## ドメイン名
- インターネット上のネットワークを一意に識別するための名称  
  ex.) 違うマンションなのに住所が一緒だと困る
- http://www. **hoge.co.jp**  // 太字部分が該当

## ホスト名
- インターネット上のネットワークのホストを一意に識別するための名称  
  ex.） 同じマンションで部屋番号が一緒だと困る
- http:// **www** .hoge.co.jp // 太字部分が該当

## FQDN(Fully Qualified Domain Name)=完全修飾ドメイン名
- ドメイン名とホスト名を繋げた文字列のこと
- http:// **www.hoge.co.jp** // 太字部分が該当
