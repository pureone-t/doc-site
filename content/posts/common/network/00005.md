# Mac (OSX) SSH接続
## 公開鍵認証
```
$ ssh <IP_ADDRESS|HOST_NAME> -l USER_NAME -i ~/.ssh/id_rsa
```

```
$ vi ~/.ssh/config
host HOGE
  hostname XX.XX.XX.XX
  user USER_NAME
  IdentityFile ~/.ssh/id_rsa
$ ssh HOGE
```
