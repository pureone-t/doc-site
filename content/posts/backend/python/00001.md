# Python インストール
## 構成
```
Homebrew
 |- python3
 |- pip3
     |- python3で使用するライブラリ...
 |- etc...
```

## Python
```
$ brew install python3
$ witch python3
$ witch pip3
```

## Libraries
```
$ pip3 install numpy  # 線形代数
$ pip3 install scipy  # 数式処理
$ pip3 install matplotlib   # 描画
$ pip3 install pandas  # データ操作
```
