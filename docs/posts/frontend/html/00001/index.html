HTML5が使われているか確認する方法
2018-03-29 (Thu) · 15 words
HTML5

HTML5 markup detector

  • HTML5 markup detector をインストール
  • HTML5で書かれたサイトにアクセスすると、ロケーションバーに緑色で 5と表示される。

仕様

  • HTML5のdoctypeの書き方が <!doctype html> であることから表示中のページのdoctypeがそれに該当するかどうかで判定
  • アドレスバーの右端に”マル５”と表示してくれるもの
