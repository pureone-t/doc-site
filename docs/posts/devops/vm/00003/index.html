Cyberduck インストール

  1. ダウンロード https://cyberduck.io/

    • 「Download Cyberduck for Mac.」

  2. インストール

    • 「Zipファイルをダブルクリック」
    • 「アプリケーションをアプリケーションフォルダへ移動」

  3. 環境設定

    1. ブラウザ　一般の「’.‘で始まるファイルを表示」「ダブルクリックしたファイルを外部エディタで開く」にチェック
    2. 外部エディタ　

接続
